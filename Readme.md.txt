# IPL Winner Prediction 🏏📊

A Machine Learning project to predict match winners in the Indian Premier League (IPL) using historical data.

## 🔍 Problem Statement

Using team-level features and past performance data to predict the match outcome (win/loss), score margins, and number of wickets left.

## 🧠 ML Models Used

- Logistic Regression  
- Random Forest Classifier  


## 📁 Project Structure

IPL_Winner_Prediction_ML/ ├── datasets/ │ ├── matches.csv │ └── [deliveries.csv not included due to size] ├── ipl_predictor.py ├── requirements.txt └── README.md

## 📉 Dataset

- `matches.csv`: contains match-level info from 2008–2024
- `deliveries.csv`: not included due to large size.  
  🔗 [Download from Kaggle](https://www.kaggle.com/datasets/vishnudhakad/ipl-complete-dataset-2008-2022) and place it inside the `datasets/` folder manually.
